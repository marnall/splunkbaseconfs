**Description:**
The CCX Add-on for NetSuite Products looks to provide a single field extraction bundle for NetSuite logs.
This TA was built using a large dataset and endeavours to be the most CIM compliant comprehensive field extraction TA available for NetSuite.

Fully compatible with Splunk Enterprise and Splunk Cloud, built by an Ops team for Ops teams.

**Features:**

- This TA currently supports logtypes tagged under the following CIM datamodels: Authentication, and Change.
