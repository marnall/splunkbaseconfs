Summary:

The Enrich app can enrich the Splunk notable events using security feeds and then send those events to ticketing system and database for prediction.
Users can modify the enrichment and ticketing integrations with their own approved feeds.

Advantages:

•	To perform the proper investigation

•	To identify the actual risk associated with the notable events

•	Enrichment and incident creation with third party applications

•	Database integration to perform the analysis of history data and make prediction using ML\AI algorithms


Future Update:

•	Enable more threat feeds to accurate the risk and urgency
