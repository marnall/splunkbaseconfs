# SentryWire App for Splunk

## Overview
The SentryWire App for Splunk provides a simple interface for analysts to easily access the advanced search capabilities of their SentWire instance from within Splunk. 

*External Sources Notice*  
This app will make external API calls to your SentryWire host as well as providing pivot links for search information such as status, metadata, pcap data, and relevant object.

## Installation

### via Splunkbase
1. In the splunk UI, click `Apps`
2. From the dropdown menu, click `Manage Apps`
3. After the local app page has loaded, click `Browse more apps`
4. Search keyword `sentrywire`
5. Click `Install`  

### via .spl File
1. In the splunk UI, click `Apps`
2. From the dropdown menu, click `Manage Apps`
3. After the local app page has loaded, click `Install app from file`
4. Use the file browser to select `sentrywire.spl`
5. Check the `Upgrade app` option
6. Click `Upload`  

*Note: You will not be prompted by any additional setup pages after app installation*
