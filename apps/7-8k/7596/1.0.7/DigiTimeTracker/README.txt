Summary:

The DigiTimeTracker app creates daily user authentication report to track login and logout information.

Report will be generated and stored inside data\Files path under the app folder.

Advantages:

•	To track the duration of user activities

•	Creates attendance report based on first login and last logout

•	User information from LDAP can be added to the report

•	To track windows lock and unlock activities

Future Update:

•	Enable leave details based on lookup

•	Automate the visualization of the result in Splunk
