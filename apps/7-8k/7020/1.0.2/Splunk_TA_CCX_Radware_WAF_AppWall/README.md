**Description:**
The CCX Add-on for Radware WAF AppWall looks to provide a single field extraction bundle for Radware WAF AppWall logs ingested via syslog.
This TA was built using a large dataset and endeavours to be the most CIM compliant comprehensive field extraction TA available for Radware WAF AppWall.

Fully compatible with Splunk Enterprise and Splunk Cloud, built by an Ops team for Ops teams.

**Features:**
- This TA currently supports logtypes tagged under the following CIM datamodels: Alert, IDS, Network Traffic, and Web.
