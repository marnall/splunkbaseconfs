**Description:**
The CCX Extensions for Axonius Add-on looks to provide field extraction and CIM compliance for Axonius log sources captured via the Add-on Axonius Add-On for Splunk.

This Technical Add-on does not replace the public Splunk Axonius Add-On for Splunk (https://splunkbase.splunk.com/app/5740) but works as an additonal extension to be deployed on Search Heads (only).

Currently this add-on provides field extraction and CIM compliance for sourcetypes "axonius_saved_query".


Fully compatible with Splunk Enterprise and Splunk Cloud, built by an Ops team for Ops teams.

**Features:**
- This TA currently supports logtypes tagged under the following CIM datamodels: Inventory
