# The Splunk Upgrader App

This app was developed by Tomas Chase

# Introduction

The purpose of this app is to allow users to easily ugrade their standalone Splunk instances. The current version is not to be used for clustered environments. It is only meant to be used on standalone monitoring consoles, deployment servers, license managers, searche heads, indexers, and heavy forwarders.

# Supported Versions

The current version will require the application to be installed on the monitoring console. The software will be delivered to instances through the use of a mini deployment server configured on the monitoring console. This can be removed once the upgrade process is completed.
