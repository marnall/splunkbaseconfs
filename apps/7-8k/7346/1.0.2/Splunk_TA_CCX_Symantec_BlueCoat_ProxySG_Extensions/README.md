**Description:**

CCX Security Operations has taken it upon ourselves to improve the existing non-public Splunk Add-on "Symantec ProxySG App for Splunk" (https://splunkbase.splunk.com/app/2815) as to ensure it is as CIM compliant as possible.

This TA does not replace the official  Splunk Add-on for "Symantec ProxySG App for Splunk", but works as an additional extension to be deployed on Search Heads (only).

Currently this add-on provides additional field extraction and CIM compliance for sourcetype:

- bluecoat:proxysg:access:kv


Fully compatible with Splunk Enterprise and Splunk Cloud, built by an Ops team for Ops teams.

**Features:**

- This TA currently supports logtypes tagged under the following CIM datamodels: Web.
