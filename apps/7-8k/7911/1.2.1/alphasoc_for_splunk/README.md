# AlphaSOC for Splunk

AlphaSOC for Splunk is an application that integrates AlphaSOC's security analytics with Splunk's data platform, providing enhanced threat detection capabilities for security teams. AlphaSOC processes logs and detects suspicious activities that might indicate security threats, offering visualization dashboards and alerting mechanisms for security operations.

## Documentation

Detailed documentation and configuration instructions can be found at [docs.alphasoc.com](https://docs.alphasoc.com).
