Steps to build tar file:

1. Build command [run in the directory where package folder is present.] : 
    ucc-gen build --ta-version <version_number>
2. Package command [This would generate a tar file in the directory where package folder is present. This tar file can be installed on splunk local instance.]: 
    ucc-gen package --path output/* 
    
Documentation can be found on Splunkbase and GitHub!
