# GIB DRP Splunk Integration.

- This is the initial version of the fully refactored DRP application for Splunk ES/HF/Cloud;
- Replaces legacy DRP Splunk app (will be removed from Splunkbase soon).

### v1.0.1
- Improved configuration parameter names.

### v1.1.1

- Fixed a bug where different accounts referred to the same seqUpdate;
- Added support for selecting an account when running the gibgetbrands, gibgetsubscriptions, and gibsetstatus commands.
