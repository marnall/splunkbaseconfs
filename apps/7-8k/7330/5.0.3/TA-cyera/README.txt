Cyera Technology Add-on for Splunk

Retrieves datastore inventory, security issues, classifications, events, and audit logs
from the Cyera REST API. Features orchestrated data collection with rate-limit coordination,
CIM-mapped field extractions, interactive dashboards, and automated CI/CD publishing.

Release lines:
- 5.x  : Python 3.13 build for Splunk Enterprise/Cloud 10.2 and later.
- 4.1.x: Python 3.9/3.13 dual-runtime build for Splunk 8.0 through 10.1.

For setup instructions and release notes, visit:
https://github.com/cyeragit/CyeraTA/releases
