Welcome to Twilio App For Splunk
Introducing the Twilio App for Splunk – This Splunk app empowers you to harness the power of Splunk to gain valuable insights into your Twilio account activity

Dashboard Overview
This extension enhances the presentation of Twilio Message Insights within Splunk, offering comprehensive visual representations for both Call Insights and Events Insights data.

Experience a streamlined view of Twilio Message, Call Insights, and Events Insights through the Twilio App for Splunk. This powerful dashboard provides insightful visualizations, enabling a comprehensive analysis of your Twilio communication data within the Splunk platform

To leverage the Twilio App for Splunk seamlessly, it's essential to first download and integrate the Twilio addon specifically designed for Splunk. This addon establishes a vital connection, facilitating the smooth utilization of Twilio's functionalities within the Splunk environment.

Download Twilio Addon for Splunk link

Macros
App is using following Macros to populate data into dashboards

twilio_messages
twilio_calls
twilio_events
Users can specify index name in macros to pull the data.
