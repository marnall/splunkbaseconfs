**Introduction:**

The SysCloud App for Splunk presents visualizations of backup, restore, export trends and utilized storage, highlighting data insights over specified periods.

**Prerequisites:**

Ensure that the SysCloud Add-on for Splunk is installed and properly set up.

**Installation:**

To install SysCloud App for Splunk directly from the Splunkbase, you can navigate to the "Find More Apps" section in the Splunk Home Dashboard.

Alternatively, you can follow these steps to install the add-on from the bundle:

- Download the App package.
- From the Splunk UI (User Interface), navigate to Apps -> Manage Apps.
- In the top right corner, select "Install app from file."
- Choose the file and upload the App package.

Follow the prompts to complete the installation.

**Use Cases:**

Monitor your backups, restores, and exports trends, along with storage utilization over time. Gain valuable insights into your backup health.
