# ThreatMon Threat Intelligence App for Splunk

A modern threat intelligence add-on that collects IOCs from ThreatMon platform.

## Features

- Real-time IOC collection from ThreatMon API
- Modern dashboard with threat statistics
- KV store integration for historical data
- Professional UI with glassmorphism design

## Installation

1. Install the app in Splunk
2. Configure ThreatMon credentials via Setup
3. Enable the modular input
