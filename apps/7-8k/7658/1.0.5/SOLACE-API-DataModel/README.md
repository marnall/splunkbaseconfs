Solsys, through its Solsyslabs R&D Group, has developed an API CIM (Common Information Model) data model for Splunk, which we wanted to donate to the Splunk Community.  Many organizations leverage APIs to collect, standardize, and analyze data across various systems as API's become the digital economy.  

Use Case Example:

Imagine managing a cloud platform with multiple authentication, billing, and user management APIs flowing through an API Gateway. An API CIM data model can unify these disparate data streams, allowing for a centralized view of user activity, API performance, and potential security threats in Splunk.
