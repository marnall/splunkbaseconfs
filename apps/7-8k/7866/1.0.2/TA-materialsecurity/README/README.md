# Material Security Add‑on for Splunk

Normalizes Material Security *Issue* alerts via Splunk HEC and maps them to CIM Change, Email, and Risk models.

## Installation
1. Upload **TA-materialsecurity-1.0.2.tar.gz** to Splunk (Apps ▸ Manage Apps ▸ Install app from file).
2. Restart Splunk if prompted.
3. Configure an HEC token and Material webhook as per documentation.

## Supported sourcetypes
- `material:issue:*`
