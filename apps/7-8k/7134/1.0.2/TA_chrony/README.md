# TA_chrony
**Add-on for chrony logs**

TA_chrony is an add-on for use on Linux systems that produce logs for [chrony](https://chrony-project.org/), a time synchronization client and server daemon based on [NTP](https://ntp.org/).

TA_chrony organizes chrony logs into source types depending on the log type. It also provides field extractions and enrichments that simplify analysis of client and server performance.

TA_chrony should be installed on indexers, search heads and forwarders.
