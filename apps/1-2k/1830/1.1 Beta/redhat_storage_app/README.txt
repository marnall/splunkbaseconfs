Red Hat Storage App for Splunk Enterprise v1.1 Beta


Red Hat Storage App for Splunk Enterprise v1.1 Beta provides the fixes and features below to better support your monitoring and reporting needs surrounding your Red Hat Storage Server installation. Note: for system requirements, installation, and configuration please refer to the "Red Hat Storage App for Splunk Enterprise Version 1.1 Beta Installation Guide.pdf" which is located in the appserver/static directory in this app.



Features:

• A drop-down form that allows RHS app users to select ethernet interfaces has been added to the Average Network Throughput panel located on the Red Hat Storage Server Overview dashboard and to the Average Outgoing/Incoming Network Traffic by Host panels on Red Hat Storage Server Health Drill Down dashboards.  

• The first cluster and/or only cluster is now the default value in the drop-down upon dashboard landing.  

Fixes:

• The NFS panel on the Red Hat Storage Overview dashboard has been updated to accurately reflect the aggregate of NFS connections.

• The submit button is now enabled. 

• The CPU utilization and network throughput panels on the Red Hat Storage Overview dashboard has been updated to accurately reflect cpu utilization.

• The incoming/outgoing network traffic panels have been updated to accurately reflect the average network traffic.
