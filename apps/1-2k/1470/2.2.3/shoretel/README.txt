# README for the Splunk for Shoretel app. 

Documentation 
For all documentation see:
https://sideviewapps.com/apps/splunk-for-shoretel/

Requirements 
  Splunk Enterprise 6.4 or higher (DB Connect 3.x compatible)
  
  SEARCH HEAD(S)
    Sideview Utils is required, version 3.3.15 or higher https://sideviewapps.com/apps/sideview-utils/
    DB Connect 3.1.X or newer required, unless on a HF
    
  INDEXER(S)
    Indexes need created - "shoretel"

    FORWARDER(S)
    HF required for DB Connect 3.1.X unless DBX is installed on the SH.
