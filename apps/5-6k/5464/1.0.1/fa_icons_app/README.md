## Overview

This app provides a gallery of all the icons in fontawesome. Once you find an icon you like, select it and the app will generate a code snippet to include in your dashboard.
An example is included of using the icons inline for an application of monitoring sensor's battery levels. The battery icons are added based on battery levels.

## Setup

Untar the release to your $SPLUNK_HOME/etc/apps directory
