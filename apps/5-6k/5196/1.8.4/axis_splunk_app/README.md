The Axis Security Splunk application adds granular real-time information about users, applications and access policy. 
The Axis app automatically pulls Axis Security access and audit data into Splunk.
This data includes insights revealed by the App Axis Cloud, which can also be integrated into real-time dashboards and reports.
 
Once added to Splunk, application updates are automatic, initiated when a new version is available.

To stream Axis Security logs with the Splunk application, please follow the instructions in the “Log streaming” screen on the Axis Security Management Console.
