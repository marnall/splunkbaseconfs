## [2.0.0] - 2021-05-17
### Added
- Custom app.manifest to include privacy policy, license, specify target workloads,
tasks, supported deployments, incompatible apps.
- Support for Splunk Enterprise
- Configuration option to identify whether Enterprise Security is installed or not
- New collections.conf and transforms.conf for non-ES deployments
- Workflow actions to include research_in_trustar

### Changed
- Modified app name to TruSTAR App for Splunk Enterprise and Enterprise Security
- Version bump to 2.0.0
- Modified programmatic name to trustar_unified
- Updated TruSTAR HTTP headers to reflect updated version
- Refactored context to include ioc mappings
- SHCluster configuration options in server.conf
- changed modinput to use "single_instance_mode"
- changed modinput config page to allow user to specify which kvstore group to
submit observables to.
