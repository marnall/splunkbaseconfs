# Slackbot+ Add-On for Splunk

A massively improved version of my previous [Slack Webhook Alert Action](https://github.com/sodle/TA-slack-webhook-alert)

## Documentation
Install the Add-On to your Splunk search head. Configuration instructions are found in the Add-On's SplunkWeb UI.
