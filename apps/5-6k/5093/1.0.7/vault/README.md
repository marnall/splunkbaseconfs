# HashiCorp Vault App For Splunk

This Splunk® App helps you gain insights into the operation and usage of your 
HashiCorp Vault clusters.  Use it to better understand how Vault is being used, prevent and 
diagnose operational issues, and improve how you secure your organization.

## Installation instructions

Please consult the [Monitor Telemetry & Audit Device Log Data with Splunk](https://learn.hashicorp.com/vault/monitoring/monitor-telemetry-audit-splunk#splunk-app) guide for information
about how to set up Vault telemetry for use with this app.
