This application was developed by Cyberfox, Astana, Kazakhstan, as a ready-made solution for the rapid deployment of monitoring and analysis of Squid proxy logs. It is designed for quick visualization of key events and simplified work with saved searches so that you can start analyzing immediately after installation.

Proxy Session Explorer, Proxy Monitoring (Errors), and Proxy Analytics are three ready—made dashboards for quick and visual analysis of Squid proxy logs: the first provides a detailed analysis of sessions and connections, the second focuses on errors and incidents, and the third provides aggregated analytics on URLs, traffic, and key metrics.

All application searches use the idx_squid macro (by default it points to index=squid), which facilitates reconfiguration: if necessary, it is enough to change the definition of the macro, and all dashboards will start working with the new index.
