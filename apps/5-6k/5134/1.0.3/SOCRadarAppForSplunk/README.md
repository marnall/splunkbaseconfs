# SOCRadar App For Splunk

SOCRadar is a cloud-based, AI-powered Digital Risk Protection Platform enhanced by cyber threat intelligence capabilities.
SOCRadar App For Splunk provides an option to visualize SOCRadar Incidents and API Data.

## Requisites

1. A SOCRadar Plan (Professional/Enterprise) with access to the SOCRadar APIs.
2. Splunk index named "socradar_incidents"
