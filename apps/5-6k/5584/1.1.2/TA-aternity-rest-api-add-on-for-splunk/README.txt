Aternity REST-API add-on for Splunk helps in pulling insights from Aternity Rest APIs in Splunk.

Steps to setup config
--------------------------------------
How to input aternity_domain_name?
Example: If your URL is "https://my-aternity.aternity.com" ,  then only input "my-aternity" in this field.

How to input rest_api_name?
Quick-Tip: For the latest version of the Aternity APIs list please visit "https://help.aternity.com/restapi"
Example Rest_api name looks like "CONNECTION_TEST" OR "DEVICE_INVENTORY" OR "DEVICE_HEALTH_RAW" etc.

What is a global_account?
Username (must have the OData REST API role) and its password, you can find this by logging into Aternity >> selecting User icon > REST API Access.
