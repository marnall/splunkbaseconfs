## ITSI Content Pack - Splunk Observability Cloud

The IT Service Intelligence (ITSI) Content Pack for Splunk Observability Cloud bridges the data gap between ITSI and Splunk Observability Cloud by providing the functionality of Splunk Synthetic Monitoring, Splunk Infrastructure Monitoring, and Splunk Application Performance Monitoring in a single view. It also includes Splunk Real User Monitoring (RUM) and RUM Mobile to monitor web properties, apps, and cloud footprint.

For installation and configuration instructions, see [Install and configure the Content Pack for Splunk Observability Cloud
](https://docs.splunk.com/Documentation/CPObservability/latest/CP/Install).

**Note**: This content pack requires that you first install the [Splunk Infrastructure Monitoring Add-on](https://splunkbase.splunk.com/app/5247).
