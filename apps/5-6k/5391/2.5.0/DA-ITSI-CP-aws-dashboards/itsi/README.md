## ITSI Content Pack

The IT Service Intelligence (ITSI) Content Pack for AWS Dashboards and Reports gives you critical operational insight into your Amazon Web Services accounts. It also enables faster time to value for AWS monitoring customers by providing an easier way to analyse and understand data by AWS account and regions for each of the OOTB entity types like AWS EC2, ELB, EBS, and Lambda Functions.

For installation and configuration instructions, see [Install and configure the Content Pack for AWS Dashboards and Reports](https://docs.splunk.com/Documentation/CPAWSDash/latest/CP/Install).

**Note**: This content pack requires that you first install and Configure the [Splunk Add-on for Amazon Web Services](https://splunkbase.splunk.com/app/1876/).
