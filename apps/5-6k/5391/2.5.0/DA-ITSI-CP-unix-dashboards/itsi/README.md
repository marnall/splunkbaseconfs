The Content Pack for Unix Dashboards and Reports provides visibility Into Operational Health and Performance and Resource Utilization Analytics. Use the dashboards along with performance metrics monitored in IT Essentials Work to get an end-to-end picture of your Unix environment

For installation and configuration instructions, see [Install the Content Pack for Unix Dashboards and Reports](https://docs.splunk.com/Documentation/CPUnixDash/latest/CP/Install).

**Note**: This content pack requires that you first install the [Splunk Add-on for Unix and Linux](https://splunkbase.splunk.com/app/833/).
