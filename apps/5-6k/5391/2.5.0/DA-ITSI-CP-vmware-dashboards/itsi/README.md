## ITSI Content Pack

Provide an overview of this Content Pack.

The IT Service Intelligence (ITSI) Content Pack for VMware Dashboards and Reports is a collection of dashboards and their Knowledge Objects that provides insights about the capacity of clusters, hosts, and allows administrators to plan for allocating resources across different VMware infrastructure components.
The dashboards leverage data collected using Splunk Add-on for VMware Metrics. With the Content Pack and apps installed customers can monitor performance of VMware and manage capacity of VMware resources as well.

For installation and configuration instructions, see [Install and configure the Content Pack for VMware Dashboards and Reports](https://docs.splunk.com/Documentation/ITSICP/current/Config/ConfigVMWDash).

**Note**: This content pack requires that you first install and Configure the [Splunk Add-on for VMware Metrics](https://splunkbase.splunk.com/app/5089/).
