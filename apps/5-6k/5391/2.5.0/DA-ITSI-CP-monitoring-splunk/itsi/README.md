    
The IT Service Intelligence (ITSI) Content Pack for Monitoring Splunk as a Service helps you model the different pieces of your Splunk 
Enterprise environment, including search heads, indexers, and search head clusters. Use the preconfigured KPIs to monitor service health, perform root cause analysis,
and receive important alerts when things change.

For installation and configuration instructions, see [Install and configure the Content Pack for Monitoring Splunk as a Service](https://docs.splunk.com/Documentation/CPSaaSMon/latest/CP/Install).
