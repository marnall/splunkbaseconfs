## ITSI Content Pack

The IT Service Intelligence (ITSI) Content Pack for NetApp Data ONTAP Dashboards and Reports provide real-time and historical visibility into the performance and configuration of your NetApp storage infrastructure. Use this Content Pack with the Splunk Add-on for NetApp Data ONTAP to collect NetApp API data from one or more NetApp FAS controllers.

For installation and configuration instructions, see [Install and configure the Content Pack for NetApp Data ONTAP Dashboards and Reports](https://docs.splunk.com/Documentation/ITSICP/current/Config/ConfigNetApp).

**Note**: This content pack requires that you first install and Configure the [Splunk Add-on for NetApp Data ONTAP Dashboards and Reports](https://splunkbase.splunk.com/app/3418/).
