## ITSI Content Pack - Splunk AppDynamics

The Content Pack for Splunk AppDynamics normalizes data from Splunk AppDynamics providing an easy way to visualize, investigate and understand Key Performance Indicators (KPIs) and Services. This content pack provides Services, KPIs and appdynamics dashboards that utilize industry standard metrics like Availability, Response Time, Throughput and Error Rates. It calculates Health Scores for each of your applications providing proactive monitoring in a consistent fashion regardless of APM tool.  If your organization uses different tools for different applications or you're migrating from one vendor to another this content pack is for you. 

For installation and configuration instructions see [Install and configure the Content Pack for Splunk AppDynamics](link)

**Note**:  This content pack uses data from below Add-On:
* [Splunk Add-On for Splunk AppDynamics](https://splunkbase.splunk.com/app/3471/)
