# ITSI Content Pack for Cisco Data Center

The IT Service Intelligence (ITSI) Content Pack for Cisco Data Center provides the scaffolding to onboard Cisco Nexus Dashboard data into ITSI. Use it as the foundation for creating entities, services, and visualizations that track the health of your Cisco data center infrastructure.

For installation and configuration instructions see [the content pack documentation](https://quickdraw.splunk.com/redirect/?product=CPCiscoDataCenter&location=install&version=current)

**Note**:  This content pack uses data from below Add-Ons:
* [Cisco DC Networking App for Splunk](https://splunkbase.splunk.com/app/7777/)
