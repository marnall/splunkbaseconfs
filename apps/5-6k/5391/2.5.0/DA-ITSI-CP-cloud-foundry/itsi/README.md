    
The IT Service Intelligence (ITSI) Content Pack for Monitoring Pivotal Cloud Foundry provides the elements necessary for monitoring 
your Pivotal Cloud Foundry (PCF) deployment. The content pack includes several preconfigured services that represent your PCF server group 
and glass tables that model your PCF environment.

For installation and configuration instructions, see [Install and configure the Content Pack for Pivotal Cloud Foundry](https://docs.splunk.com/Documentation/CPPCFMon/latest/CP/Install).
