    
The IT Service Intelligence (ITSI) Content Pack for Shared IT Infrastructure Components supports approaches for mapping service dependencies across the applications and departments
within your IT environment. It contains sample services, a custom service analyzer view, and a glass table. 

For installation and configuration instructions, see [Install and configure the Content Pack for Shared IT Infrastructure Components](https://docs.splunk.com/Documentation/CPSharedInfra/latest/CP/Install).
