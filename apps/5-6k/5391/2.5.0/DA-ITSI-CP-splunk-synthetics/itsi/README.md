## ITSI Content Pack - Splunk Synthetic Monitoring

The Content Pack for Splunk Synthetics provides an easy way to visualize, investigate and understand Key Performance Indicators (KPIs) and Services based on your Synthetic Checks.  Using the Health Scores and KPIs in the Service Analyzer you can quickly identify problematic areas and drill down to a specific run and a specific synthetic check and launch in context to see all of the details in Splunk Synthetics.    

For installation and configuration instructions see [Install and configure the Content Pack for Splunk Synthetic Monitoring](https://docs.splunk.com/Documentation/CPSyntheticMon/latest/CP/Install)

**Note**:  This content pack requires the [Splunk Synthetics Monitoring Add-on](https://splunkbase.splunk.com/app/5608/).
