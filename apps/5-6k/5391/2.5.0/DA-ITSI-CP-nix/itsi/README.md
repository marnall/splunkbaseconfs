   
The IT Service Intelligence (ITSI) Content Pack for Monitoring Unix and Linux provides the elements necessary to monitor your OS-level health related to Linux and certain types of Unix servers. 
It includes a service template and KPIs for monitoring OS health metrics for CPU, memory, disk, and network.

For installation and configuration instructions, see [Install and configure the Content Pack for Monitoring Unix and Linux](https://docs.splunk.com/Documentation/ITSICP/current/Config/ConfigNix).

**Note**: This content pack requires that you first install the [Add-on for Unix and Linux](https://splunkbase.splunk.com/app/833/).
