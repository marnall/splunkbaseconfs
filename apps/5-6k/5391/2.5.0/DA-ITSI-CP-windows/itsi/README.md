The IT Service Intelligence (ITSI) Content Pack for Monitoring Microsoft Windows provides the elements necessary for monitoring your OS health related
to Windows servers. It includes KPI base searches as well as a service template for monitoring OS health metrics for CPU, memory, disk, and network. 

For installation and configuration instructions, see [Install and configure the Content Pack for Monitoring Microsoft Windows](https://docs.splunk.com/Documentation/CPWindowsMon/latest/CP/Install).

**Note**: This content pack requires that you first install the [Add-on for Microsoft Windows](https://splunkbase.splunk.com/app/742/).
