
The IT Service Intelligence (ITSI) Content Pack for VMware Monitoring provides the elements necessary to monitor the performance of the main components
in a VMware vSphere environment: ESXi servers, virtual machines, and datastores. It includes a set of KPI base searches, service templates, and a sample service topology tree.

For installation and configuration instructions, see [Install and configure the Content Pack for VMware Monitoring](https://docs.splunk.com/Documentation/CPVMwareMon/latest/CP/Install).

**Note**: This content pack requires that you first install the [Splunk Add-on for VMware Metrics](https://splunkbase.splunk.com/app/5089/).
