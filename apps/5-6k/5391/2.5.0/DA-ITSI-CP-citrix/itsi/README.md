   
The IT Service Intelligence (ITSI) Content Pack for Monitoring Citrix provides a quick way to monitor your Citrix virtual apps and desktop infrastructure. 
The content pack includes predefined saved searches paired with data from Citrix XenDesktop technical add-ons. Each service template comes with
key performance indicators (KPIs) that provide fast visibility into your critical systems.

For installation and configuration instructions, see [Install and configure the Content Pack for Monitoring Citrix](https://docs.splunk.com/Documentation/CPCitrixMon/latest/CP/Install).

**Note**: This content pack requires that you first install the [Add-on for the Content Pack for Monitoring Citrix](https://splunkbase.splunk.com/app/5103/), otherwise the content pack installation fails.
