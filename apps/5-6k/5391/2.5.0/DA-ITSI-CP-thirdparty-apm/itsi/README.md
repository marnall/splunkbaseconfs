## ITSI Third Party APM

The Content Pack for Third Party APM normalizes data from Dynatrace and New Relic providing an easy way to visualize, investigate and understand Key Performance Indicators (KPIs) and Services. This content pack provides Services, KPIs, Glass Tables and vendor specific dashboards that utilize industry standard metrics like Apdex, Availability, Response Time, Throughput and Error Rates. It calculates Health Scores for each of your applications providing proactive monitoring in a consistent fashion regardless of APM tool. If your organization uses different tools for different applications or you're migrating from one vendor to another this content pack is for you. 

For installation and configuration instructions see [Install and configure the Content Pack for Third Party APM](https://docs.splunk.com/Documentation/CPAPM/latest/CP/Install)

**Note**: This content pack uses data from the following 2 Add-Ons:
* [Splunk Add-On for New Relic](https://splunkbase.splunk.com/app/3465/)
* [Dynatrace Add-On for Splunk](https://splunkbase.splunk.com/app/3969/)
