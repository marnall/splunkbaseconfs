   
The Content Pack for ITSI Monitoring and Alerting helps ITSI administrators and service owners translate service and KPI health into 
meaningful episodes and take action when necessary. The content pack provides a set of correlation searches and aggregation policies which, 
when enabled, produce meaningful and actionable alerts.

For installation and configuration instructions, see [Install and configure the Content Pack for ITSI Monitoring and Alerting](https://docs.splunk.com/Documentation/ITSICP/current/Config/ConfigMA).
