**Description:**

The CCX Unified Splunk Add-on for Trend Micro looks to provide a single field extraction bundle for all Trend Micro endpoint protection technologies.
This TA was built using a large dataset and endeavours to be the most CIM compliant comprehensive field extraction TA available for;
- Deep Security
- Apex One
- Apex Central
- Control Manager
- Cloud One

Fully compatible with Splunk Enterprise and Splunk Cloud, built by an Ops team for Ops teams.

**Features:**

This TA currently supports logtypes tagged under the following CIM datamodels: Alert, Change, Malware, Endpoint, Data Loss Prevention(DLP), Email(Email filter), Intrusion Detection (IDS), Network Traffic, Web and Inventory.
