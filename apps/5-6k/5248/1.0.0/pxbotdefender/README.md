# PerimeterX Bot Defender App for Splunk

This app is designed to work with the PerimeterX Bot Defender Data Export for Splunk.

## Prerequisites
- Follow the [PerimeterX documentation](https://docs.perimeterx.com/pxconsole/docs/data-integration-to-third-party-apps) to set up Data Export to Splunk.

## Installation
- Follow the [PerimeterX documentation](https://docs.perimeterx.com/pxconsole/docs/splunk-integration) to Use the PerimeterX Bot Defender App for Splunk.

## Customization
- After installing the PerimeterX Bot Defender App for Splunk, you may customize the visualizations by changing the defaults, or updating the visualization types.
- These changes will be local to your installation of the Splunk app and need to be done per installation of Splunk.

## Support
- Email support at support@perimeterx.com or review the [PerimeterX documentation](https://docs.perimeterx.com/pxconsole/docs) for support with the App or with setting up Data Export to Splunk.
