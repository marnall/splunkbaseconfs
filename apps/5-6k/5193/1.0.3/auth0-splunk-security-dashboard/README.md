# Auth0 App for Splunk

This app is designed to work with the Auth0 Log Streaming Integration for Splunk.

## Prerequisites
- Follow the [Auth0 documentation](https://auth0.com/docs/logs/export-log-events-with-log-streaming/splunk-event-logs) to set up Log Streaming to Splunk.

## Installation
- Follow the [Auth0 documentation](https://auth0.com/docs/logs/export-log-events-with-log-streaming/splunk-dashboard) to Use the Auth0 App for Splunk.

## Customization
- After installing the Auth0 App for Splunk, you may customize the visualizations by changing the defaults, or updating the visualization type.
- However, these changes will be local to your installation of the Splunk app and would need to be done per installation of Splunk.

## Support
- Visit https://support.auth0.com for support with the App or with setting up Log Streaming to Splunk.
