# Artica TA Add-on
Artica TA Add-on provides CIM compliant field extractions, event types and tags for Artica services.

This Add-on is an integration of several other Add-ons and adapted to Artica needs, such as:
* Splunk Add-on for Squid Proxy - https://splunkbase.splunk.com/app/2965/
* Splunk Add-on for NGINX - https://splunkbase.splunk.com/app/3258/
* Splunk TA for Suricata - https://github.com/jwindley-splunk/Splunk_TA_suricata
* Splunk Add-on for HAProxy - https://splunkbase.splunk.com/app/3135/
* TA-linux_iptables - https://splunkbase.splunk.com/app/4490
## Supported Services
* SQUID PROXY
* NGINX
* AUTH
* SSH
* HAPROXY
* HACLUSTER
* FIREWALL
* SURICATA IDS
* Web Console Access information
