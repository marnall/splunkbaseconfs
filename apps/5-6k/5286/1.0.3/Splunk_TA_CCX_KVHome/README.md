**Description:**

The CCX KVHome Technical Add-on for Splunk solves the problem of large replication bundles in enterprise environments, the KVHome TA allows for any of the CCX Detection code outputs to re-home themselves into the KV Store with zero downtime or re-coding efforts.
Fully compatible with Splunk Enterprise and Splunk Cloud.
