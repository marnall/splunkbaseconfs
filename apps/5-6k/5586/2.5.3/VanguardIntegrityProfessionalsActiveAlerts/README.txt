Welcome To Vanguard Active Alerts:


Application Description: 
Vanguard Active Alerts : Vanguard Active Alerts ® for Splunk ® is a Splunk application in which 
raw-security events are transformed into actionable data that allows system/security administrators to 
make data-driven decisions. The application contains 37 different dashboards of analyzed and visualized data.


Splunk Enterprise requirements

For information about Splunk Enterprise hardware and software requirements,
see the system requirements in the Splunk Enterprise installation
documentation.

Splunk Enterprise versions
Vanguard Active Alerts for Splunk application works with Splunk Enterprise 8.05 or greater.

INSTALLATION

Install Vanguard Active Alerts Splunk application in a single-instance environment.
The following information describes how to install the application in a Splunk Enterprise deployment.

Install the application by using Splunk Web

1. Log in to the Splunk Enterprise search head.
2. From Splunk Home, click the Manage Apps cog icon to open the Apps page.
3. Click Browse more apps, locate the application in the list, or type Vanguard Active Alerts into the
   search box.
4. Provide your splunk.com credentials.
5. Accept the license terms.
6. Click Login and Install.
7. Click Done .
8. Restart Splunk services to complete the installation.

Install the application from a downloaded file

1. Log in to splunkbase.splunk.com
2. Download VanguardIntegrityProfessionalsActiveAlerts-2.4.1.tar.gz and save it to an accessible location.
3. Log in to the Splunk Enterprise search head.
4. From Splunk Home, click the Manage Apps.
5. On the Apps page, click Install app from file.
6. On the Upload app page, click the Choose file button to locate the application.
7. Click Upload.
8. Restart Splunk services to complete the installation.
