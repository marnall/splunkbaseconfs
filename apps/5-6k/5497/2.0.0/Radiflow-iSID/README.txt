Radiflow’s iSID Intrusion Detection System (IDS) for OT networks is a server-based software that analyzes the OT network traffic in order to protect against cyber threats.
iSID application for Splunk is aimed for collecting alerts and user activity events generated by iSID server.
The application collects the logs sent from iSID, analyazes them, and presents high-level summary on the main dashboard.
The application also includes built-in CIM mapping which will automatically apply in case CIM is installed.
