Druv Prism - AI-powered NL-to-SPL Query Assistant
Version: 1.1.0
Author: druvadmin
Compatible: Splunk Cloud

DESCRIPTION
Druv Prism translates natural language questions into validated Splunk SPL queries using LLM-powered schema-aware generation with continuous per-tenant learning.

INSTALLATION
1. Install the app via Splunk Web or CLI
2. Navigate to Apps > Druv Prism
3. Click "Launch Druv Prism" to open the platform

REQUIREMENTS
- Active Druv Prism account at https://prism.druv.tech
- Splunk Cloud
