This is an add-on meant for use with Viavi Observer Apex. 
To get started after installing the add-on: 
1. Open the Configuration page to configure the addon. The addon creates a HEC, which requires a name. It also requires a customizable index and sourcetype to write Apex alarms to 
2. On the Configuration page, you will be provided with a URL. Use this URL when creating a notification setting in Apex. Create an alarm in Apex that exports to the notification setting you created. Refer to ObserverDocs for more information. 
3. Apex should now be able to forward alarms to splunk. Utilize the Search and Dashboard tabs in the add-on to view the alarms.
