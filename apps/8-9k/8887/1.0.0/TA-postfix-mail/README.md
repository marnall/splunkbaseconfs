# Postfix Mail Add-on for Splunk

## Overview

The **Postfix Mail Add-on for Splunk** provides search-time field extractions, field aliases, event types, and CIM-compatible mappings for Postfix mail server logs.

The add-on is designed to parse Postfix mail logs collected from `/var/log/maillog` or equivalent mail log files and normalize them for easier searching, reporting, and dashboarding in Splunk.

Recommended sourcetype:

postfix:syslog

##Author

**Cyberfox**
 
##Version

1.0.0

##Supported Data Sources

This add-on supports Postfix mail logs generated by common Postfix components, including:

postfix/smtpd
postfix/submission/smtpd
postfix/smtp
postfix/qmgr
postfix/cleanup
postfix/bounce
postfix/smtp-amavis/smtp
amavis
Features
Field extractions for Postfix queue IDs
Sender and recipient extraction
SMTP relay, destination host, destination IP and destination port extraction
Delivery status and delivery reason parsing
SMTP delay field normalization
TLS connection parsing
Amavis scan result parsing
Amavis queue ID, message ID, mail ID, hits, size and DKIM extraction
SMTP connection failure parsing
CIM Email field aliases
Static enrichment through EVAL fields instead of lookup files
