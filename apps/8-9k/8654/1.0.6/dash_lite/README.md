# DASH Lite - Styled App Builder for Splunk

**A free preview of DASH — design CSS themes for Splunk classic XML dashboards with live preview. Includes the Cybersecurity use case with 2 dashboards.**

---

## Features

- **Visual Style Builder**: Design CSS themes through a form-based interface with live preview
- **Cybersecurity Use Case**: Preview your styles across Security Posture and Network Security dashboards
- **Color Presets**: 10 named palettes for instant color scheme application
- **WCAG Contrast Checker**: Real-time contrast ratio indicators
- **Theme Consistency Score**: 0-100 score evaluating your theme quality
- **Undo/Redo**: Full state stack with keyboard shortcuts
- **Auto-Save Drafts**: Resume where you left off

## Installation

### From Splunkbase
1. Download the latest `.spl` package from [Splunkbase](https://splunkbase.splunk.com/app/XXXXX)
2. Install via Splunk Web: **Apps > Manage Apps > Install app from file**
3. Restart Splunk

---

## Usage

1. Navigate to **Apps > DASH Lite** in Splunk Web
2. Open the **Builder** to design your dashboard style
3. Use the **Style Manager** to customize colors, fonts, logos, and backgrounds
4. Preview your styles across 2 Cybersecurity dashboards
5. Upgrade to the full version to export styled apps
