LinkShadow NDR Splunk Add-on
----------------------------
This Technology Add-on provides CIM-compliant field extractions for LinkShadow NDR CEF logs.

Installation:
1. Install this App on your Splunk Indexers (for parsing) and Search Heads (for search-time knowledge objects).
2. Configure your inputs.conf to monitor your syslog file.
3. Set the sourcetype to: linkshadow_ndr_cef
