**Description:**

CCX Security Operations has taken it upon ourselves to develop a CCX Add-on for Snowflake to provide CIM compliance coverage for logs ingested via HEC (JSON) and Generic S3 input leveraging Splunk Add-on for AWS.
This TA was built using a large dataset and endeavours to be the most CIM compliant comprehensive field extraction for Snowflake JSON logs.

**Features:**

- This TA currently supports logtypes tagged under the following CIM datamodels: Change, and Authentication.
