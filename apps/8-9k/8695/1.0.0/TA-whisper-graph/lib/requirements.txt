# Vendored Python dependencies for TA-whisper-graph

# All dependencies are pinned to exact versions (==) for Splunk Cloud Vetting:
# Cloud-Ready guidelines require third-party components to be up-to-date with
# no outstanding CVEs. Exact pins give deterministic builds and let CVE
# scanners (pip-audit, safety) flag known vulnerable versions.

# Top-level dependencies (declared by the TA)
# ---------------------------------------------------------------------------

# CVE-2026-25645 (predictable temp filenames in extract_zipped_paths).
requests==2.33.1

# urllib3 1.26.x line is on maintenance; pinned to the latest 1.26 patch.
# splunktaucclib 8.1.0 declares `urllib3 (<2)` in its METADATA, so we cannot
# move to urllib3 2.x until the upstream UCC release lifts the constraint.
#
# Known CVEs in urllib3 1.26.x (all fixed only in 2.x line):
#   - CVE-2025-50181 (PoolManager retries / redirects)
#   - CVE-2025-66418 (chained Content-Encoding decomp bomb)
#   - CVE-2025-66471 (streaming decomp bomb)
#   - CVE-2026-21441 (redirect decomp bomb, preload=False)
#
# Mitigations applied in whisper_api_client.py:
#   - requests.Session defaults (no manual PoolManager retry config).
#   - Responses are JSON payloads from the Whisper Graph API; not streamed.
#   - Per-request timeouts and bounded retry counts are enforced.
urllib3==1.26.20
