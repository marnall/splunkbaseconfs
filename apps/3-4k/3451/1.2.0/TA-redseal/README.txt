RedSeal App for Incident Investigation

Description:
RedSeal’s security analytics platform builds an accurate, up-to-date model of an organization’s entire as-built network
to visualize access paths and prioritize what to fix, to protect your most valuable assets.

Through RedSeal’s integration with Splunk, this App helps to accelerate Incident Investigation by providing security and
network access context for incidents:
    * Lists all downstream critical assets reachable from the threat source
    * Displays exact access paths from a threat source to each reachable target, and identifies the specific firewall(s)
      and configuration rules to remediate access and prevent the Incident from becoming a breach
    * Provides a dashboard to display security metrics, including your network’s ‘Digital Resiliency Score’

For documentation, see: http://www.redseal.net/files/RedSeal-Splunk App for Incident Investigation.pdf
