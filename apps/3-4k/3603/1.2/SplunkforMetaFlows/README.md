### Description ###

This Splunk application interprets system log messages generated by MetaFlows Security System's sensors. It breaks down the fields in the log messages to facilitate security analsys. It also provides links to drill down for the events on the MetaFlows Security system Web console at nsm.metaflows.com where events can be further investigated and packet payloads can be obtained.
### Documentation ###

**Installation and Getting Started:** http://www.metaflows.com/wiki/Log_Management#Splunk_App
**Support:** support@metaflows.com 

This app receives syslog messages on tcp port 3015 encapsulated using SSL. We include a generic self-signed SSL certificate in the cert directory. This certifcate was generated using the commands:
	 export OPENSSL_CONF=$SPLUNK_HOME/openssl/openssl.cnf
 	$SPLUNK_HOME/bin/genRootCA.sh -d $SPLUNKHOME/etc/apps/SplunkforMetaFlows/certs
 	$SPLUNK_HOME/bin/genSignedServerCert.sh -d $SPLUNKHOME/etc/apps/SplunkforMetaFlows/certs -n splunk -p

We recommend you generate a new SSL ceritificate using your own phassphrase by repeating the commands above.
The passphrase you enter when generating the certificate should then be placed in $SPLUNK_HOME/etc/apps/SplunkforMetaFlows/default/inputs.conf under the [SSL] stanza as:
	password = <yourpassword> 

### Install From Source ###

This app is available on [Splunkbase](http://splunkbase.splunk.com/app/3603)
Optionally, you can install the app directly.

From the directory `$SPLUNK_HOME/etc/apps/`, type the following command:

   wget https://nsm.metaflows.com/SplunkforMetaFlows.tgz
   tar xvfz SplunkforMetaFlows.tgz
