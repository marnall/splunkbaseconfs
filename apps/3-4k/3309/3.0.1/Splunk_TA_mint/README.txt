Splunk MINT Add-on
========================

OVERVIEW
--------
The Splunk MINT Add-on pulls data from the MINT Data Collector for the individual mobile app projects you set up in Splunk MINT Management Console, then lets you view your mobile app data in Splunk Enterprise.

* Author - Splunk
* Version - 3.0.1
* Compatible with:
  * Splunk Enterprise version: 7.2.x, 7.3.x, 8.0.x
  * OS: Platform independent

For more information refer, https://docs.splunk.com/Documentation/MintAddon/latest/UserGuide/AbouttheSplunkMINTAddon
