This is the add-on that downloads alerts from the Niddel v2 API into Splunk. A valid
Niddel Magnet API key is needed to use this, please e-mail us at contact@niddel.com if
you wish to start a trial.

It was built with the Splunk Add-on Builder (https://splunkbase.splunk.com/app/2962/).

Splunk customers can learn more at the knowledge base at https://support.niddel.com
using their Niddel Magnet Portal credentials.
