# Duo Splunk Connector

## App Description
Duo Splunk Connector enables Duo customers to pull their Administrator, Authentication, and Telephony logs into Splunk.
Duo Access customers will also be able to pull Endpoint information into Splunk.

To begin using the Duo Splunk Connector, follow the instructions on the Duo Overview dashboard to enable and configure
an Admin API application.


## Reporting Security Vulnerabilities

# Security Policy

## Reporting a Vulnerability
**Please do not use Github issues or pull requests to report security vulnerabilities.**

If you believe you have found a security vulnerability in Duo software, please follow our response process described at
https://duo.com/support/security-and-reliability/security-response.

## Python 2 Deprecation
Starting with release 1.2.0 of duo_splunkapp, Python 2 will not be supported.
