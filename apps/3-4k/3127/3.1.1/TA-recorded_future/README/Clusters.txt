Running the app in a Searchhead cluster
=======================================

Installing using a deployer
---------------------------

1. Using a staging machine, install the Recorded Future for Splunk ES app.
2. Remove any files in:
   $SPLUNK_HOME/etc/apps/TA-recorded_future/tmp/
   $SPLUNK_HOME/etc/apps/TA-recorded_future/lookups/
   $SPLUNK_HOME/etc/apps/TA-recorded_future/data/local/threat_intel/
3. Copy the app from the staging machine to the deployer.
4. Optionally, restart the Splunk system.
5. Deploy from the deployer.
6. On each member of the Search Head Cluster:
   a. Start the "Splunk ES Add-on for Recorded Future" app.
   b. Run "Setup" from the menu.

Install on each search head
---------------------------

It is possible to install the app on each search head. When installing
this way no particular the install is done in the normal way.
