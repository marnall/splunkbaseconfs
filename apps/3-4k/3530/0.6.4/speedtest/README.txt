Steps required to use this app:
===============================

1. Create an event index called 'speedtest'
2. Ensure the 'speedtest-cli.py' script is enabled under 'Settings > Data inputs'

Optional configuration:
=======================

By default the script is set to run every 30 minutes (1800 seconds). To change this, go to 'Settings > Data sources' and click on the 'speedtest-cli.py' script. You can then set the test interval to the desired number of seconds.
