Splidgets v.1.2
-----------------------------------------------
What is a Splidget?
-----------------------------------------------
A Splidget is a Splunk component that, in my mind, is not quite an app but does provide functionality to the end user experience in Splunk.
The idea here is that these "widgets" will provide new users ways to learn and work for efficiently within Splunk. Experienced users will also gain value.
Splidgets were designed around very simplistic principles, using as simple a process as possible to provide value to Splunk users.
-----------------------------------------------
SearchFu!
-----------------------------------------------
SearchFu! provides easy access to over 300 prebuilt searches.
I have utilized content from my search toolbox. I have gathered these searches from all my great engagements here at Splunk over the last 5 years. There is nothing you can search that can't be searched... All you need is Splunk.
These searches can be edited and added to or built up from scratch by utilizing Luke Murphey's Look up Editor.  Just click on the search and it launches in the ol’ search window.
-----------------------------------------------
Dependencies
-----------------------------------------------
This app or set of "Splunk Widgets" needs Luke Murphey’s Lookup Editor app installed for a lot of the individual workflows.
I do have error handling built in to look for the app and change the UI accordingly.
