Splunk Calendar Custom Visualization
==================
The Splunk Calendar Custom Visualization displays event statistics on a calendar.  The following views are available:

* Month
* Week
* Day

# Version History

## v1.1.1
* updates to jQuery
* updates to underscore

## v1.1.0

* Choose the default date to show.  Previously, the current date was used as a starting point.  Now, you can choose whether the first or last event returned from a Splunk query is used as the default display date.
