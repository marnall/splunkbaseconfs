## User Guide

### Concepts

This Iguana Web Analytics Support Add-on for Machine Learning is designed to help users gain insights on predictive analytics, clustering, forecasting, and anomaly detection with respect to Web Analytics data.
This SA is a support application for its base, Iguana Web Analytics App for Splunk. Hence this application should be deployed along with the base application.

### Index-Time Operations

The Iguana Web Analytics Support Add-on for Machine Learning does not perform any index-time operations.

#### Host Metadata Field Overwriting

There is no host metadata field overwriting

#### Sourcetype Metadata Field Overwriting

There is no sourcetype metadata field overwriting

### Data Inputs

This support add-on does not contain or expect any data inputs to be configured.

### Macros

There are no macros associated with this app.

### Lookups / KV Stores

There are no lookups or KV Stores associated with this app.


### Datamodels

The Iguana Web Analytics Support Add-on for Machine Learning contains zero datamodels.

### Saved Searches

There are no saved searches within this app.

### Troubleshooting

Once this SA is deployed, navigate to the now visible "Machine Learning" menu item within Iguana Web Analytics App for Splunk. In this menu, you will find four (4) sections:

Clustering

Prediction

Forecasting

Anomaly Detection

The dashboards within the above sections should show up with reports.
