## ThreatSecure Network App v1.1

**Important note:**
This app contains dashboard and reports.  No data is collected by this app.  For ThreatSecure Network Appliance data collection, install and configure the separate ***ThreatSecure Network Add-on***

## Hardware and software requirements

* Splunk 6.0+

### Prerequisites

Install and configure the ***ThreatSecure Network Add-on***

### Splunk Enterprise system requirements

Because this app runs on Splunk Enterprise, all of the Splunk Enterprise system requirements apply.

## Installation Steps

1. From the Splunk Web interface, click on **Apps > Manage Apps** to open the Apps Management page.
2. Click **Install app from file**, locate the ***ThreatSecure_TA*** .spl file, and click **Upload**.  
__Note__: ensure the “Upgrade app” checkbox is checked.

## Known issues

* If new data input is added in another window, then new tenant is not shown in the tenant dropdown of current dashboard. The new tenant will be shown only after reloading
dashboard page.
