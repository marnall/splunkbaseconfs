ServiceNow Security Operations add-on for Splunk

When Splunk is integrated with the ServiceNow Security Operations applications, you can seamlessly create security incidents or events from Splunk events, alerts, and logs. After you have downloaded the ServiceNow Security Operations add-on for Splunk from Splunkbase, you are ready to use the integration to create the desired security records.
Full documentation available at https://docs.servicenow.com/bundle/helsinki-security-management/page/product/splunk-integration/concept/secops-integration-with-splunk.html

SUPPORTED VERSIONS
* Splunk Cloud
* Splunk Enterprise 6.5
* Splunk Enterprise 6.4
* Splunk Enterprise 6.3
* Splunk 6.5
* Splunk 6.4
* Splunk 6.3

INSTALLATION
You must install and configure the ServiceNow Security Operations add?on for Splunk to link Splunk to ServiceNow. See https://docs.servicenow.com/bundle/helsinki-security-management/page/product/splunk-integration/concept/setup-splunk-integration.html for full instructions.
