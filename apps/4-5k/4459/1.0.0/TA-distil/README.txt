# README.txt

This technology add-on app is designed to provide CIM extractions and tags for Disti logs.

# Getting Started

This app has been created to work correctly with a stand-alone, distributed, and cloud installs of Splunk. You will only need this app.

## New Install

This section is to install on a centralized or stand-alone splunk setup.

1. Ensure you have syslog collection from distil.
2. Ensure the sourcetype is set as distil
3. Install TA-distil on your indexers, heavy forwarders, Syslog collectors and search heads. Restart
4. Done!
