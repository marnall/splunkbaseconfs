The purpose of this TA is to improve the user experience with Splunk with a fully standardized schema, enhance operations efficiency by lowering Splunk licensing costs, increase Splunk data retention via aggregated events, reduce Splunk hardware resource utilization, all while simultaneously improving operations’ capabilities.  This cross-pollination of features synthesizes the best of ArcSight and Splunk to provide users with a hybrid solution that is as flexible and powerful as possible.
