## System Requirements
2 Steps supports all systems that adhere to Splunk Enterprise specifications. There is however a minimum screen size due to the nature of the app that would cause users with screen sizes smaller than a typical tablet device (1024x768 px) to be unable to utilize all features.

## Installation
1. In your Splunk Enterprise web interface, click on App(s) -> Manage Apps.
2. Click on the 'Install app from file' button.
3. Select the downloaded file, Click Upload, optionally selecting Upgrade app if you are upgrading from an earlier version of the application. A restart of Splunk may be required for changes to take affect.

## License Activation / Configuration
2 Steps setup requires the purchasing of a license through [2 Steps](https://www.2steps.io/). This license should then be uploaded using the in app 'license' page to activate and configure the product to the purchased specifications.

## Troubleshooting
1. Check that the application is up-to-date. If not, follow the installation and configuration steps highlighted above.
2. Restart Splunk
3. Contact 2 Steps Support at: support@2steps.io
