This app gives you the ability to schedule auto enabling or disabling of reports which may help you to circumvent scheduled maintenance, planned upgrades, or simply weekends with not personnel coverage. A report group can be selected by choosing one or more of the following filtering criteria
- Packaging Splunk application
- Name of saved search or report
- Owner
- Status (Enabled/Disabled)

There are two kinds of group actions available
1. Permanent Changes - This action will enable or disable all filtered reports
2. Timed Changes - This action will allow you to specify a schedule during which saved searches need to be enabled or disabled. A recurrence can also be specified for timed changes. Schedules are stored in a lookup table and can be managed by App > Manage > Manage Schedules
