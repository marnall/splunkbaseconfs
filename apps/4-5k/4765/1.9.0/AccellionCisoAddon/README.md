# Accellion Ciso Addon

## Description
Accellion Ciso Addon is an addon for Accellion Ciso application in Splunk to index data received, define source type, extract any fields needed, and define CSV lookups. This addon receives data from Accellion application and provide the results to Accellion Ciso application in Splunk. The data could be from Splunk Universal Forwarder or Syslog Server.

## Installation
Here is the steps on how to install Accellion Ciso Addon if you have the tar file.
1. Go to Manage Apps.
2. Click Install app from file.
3. Click Choose file.
4. Locate the AccellionCisoAddon.tar.gz.
5. Check Upgrade app if want to upgrade.
6. Click Upload.
7. Go to Settings.
8. Choose Server controls.
9. Click Restart Splunk.
10. Choose Ok.
