TA-Splunk_Notable_Edits version: 0.5

This App includes functions to manipulate Notable Events within Splunk Enterprise Security.

Currently included functions :

updatenotableowner

Update a Notable Event to re-assign ownership to a current, valid Splunk user.

updatenotableowner <EventIDField> newOwner = <NewOwner> newOwnerField = <FieldName> newOwnerField = <FieldName>
