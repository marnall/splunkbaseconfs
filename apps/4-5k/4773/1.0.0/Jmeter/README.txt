Description:
"JMeter app for splunk" Is an app to visualize JMeter data using splunk.

In JMeter 5.1 the only way to export data in the format supported by this app is using tsv. However in the future versions we plan to more support to export data using JMeter logfiles.

You can update data input setting in default/inputs.conf
The app in configured to use macro which defaults to index=main inorder to update index please update it in defaults/macros.conf

You can find more information on how to send data from JMeter in 
/default/data/ui/html/help.html
