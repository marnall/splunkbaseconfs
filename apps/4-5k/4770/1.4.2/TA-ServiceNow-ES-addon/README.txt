ServiceNow Security Operations Event Ingestion Addon for Splunk ES
With the ServiceNow Security Operations Event Ingestion Addon for Splunk ES, you can forward notable events from Splunk ES Incident Review to ServiceNow to create security incidents in ServiceNow Security Incident Response application.  This Splunkbaase app works in conjunction with the ServiceNow app store integration located here - https://store.servicenow.com/sn_appstore_store.do#!/store/home (search for Splunk ES to find specific integration)
SUPPORTED VERSIONS
* Splunk Cloud
* Splunk 8.0
* To Use Splunk ES add-on app 1.0.3, Minimum required Python version is Python 3

INSTALLATION
You must install and configure the ServiceNow Security Operations add-on for Splunk to link Splunk to ServiceNow. This Splunkbase app works in conjunction with the ServiceNow app store integration located here - https://store.servicenow.com/sn_appstore_store.do#!/store/home (search for Splunk ES to find specific integration)
