== CyberInt Argos app for Splunk ==

== Dependencies ==
Splunk on Windows requires that that a standalone version of python version 2.7 be installed as the default python install does not contain the required modules.  

== System Requirements ==
This app has been tested to function on 7.2 of Splunk® installed on a 64 bits Windows Server 2012R2

== System Requirements ==
CyberInt Argos App for Splunk helps users to enhance their investigation capabilities.

The main purposes of this app is to serve two main business flows:
•	Incorporate an automatic validation of an IOC by querying Argos as part of the correlation flow
•	Provide a manual, ad-hoc API for a SOC engineer to query Argos directly from the Splunk dashboard.
