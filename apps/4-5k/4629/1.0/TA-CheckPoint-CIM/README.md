This TA provides Splunk Common Information Model (CIM) mappings for Check Point logs brought in using the Check Point App for Splunk and Check Point Log Exporter.

# Prequisites
- Check Point App for Splunk (https://splunkbase.splunk.com/app/4293/) should be installed and configured to bring in events using Check Point Log Exporter.
- As the Check Point App for Splunk details, Log Exporter must be set to use "splunk" format.

# Where to install
- Install TA-CheckPoint-CIM on ad-hoc and Enterprise Security (ES) search heads.

# Purpose
- TA-CheckPoint-CIM provides field aliasing, tagging, and event typing to make logs from Check Point Log Exporter (in "splunk" format) CIM compliant.
