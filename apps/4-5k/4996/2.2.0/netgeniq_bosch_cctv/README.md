About this App
--------------
People & Vehicle Analytics App for Bosch CCTV is built for Splunk® Enterprise users to quickly gain value out of powerful video analytics available on Bosch CCTV cameras. The App is designed to work with NETGENIQ FABRIC software. - a purpose built software - to extract and convert various metadata including video analytics data from CCTV and other Electronic Security systems.

NETGENIQ FABRIC software access the Bosch CCTV camera video metadata stream, extract and process metadata tags to produce event logs. These events are logged in JSON format and can be indexed into Splunk® software directly or via a Splunk® Universal Forwarder.
