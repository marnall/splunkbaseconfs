Overview
This Add-on provides valuable insights on informations added in your bookstack account.

Bookstack Add-on leverages data from Bookstack to Splunk via API Call. It uses multiple API to fetch data from Bookstack such as information about users , pages added, shelves created, attachments and books created by user.
Get the Activation key by visiting https://avotrix.com/splunkapp.php
