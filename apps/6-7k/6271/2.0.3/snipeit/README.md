# splunk-snipeit
SnipeIT Add-On for splunk

##Description
Integrates with Snipe-IT app in order to lookup user and asset inventory via Splunk.

## Version Support
Supports Splunk cloud versions 8.2 and higher.

## Configuration:
This Add-On requires a Snipe-IT API key in order to integrate with it. Navigate to `https://<splunk-cloud-deployment>splunkcloud.com/en-US/manager/snipeit/apps/local/snipeit/setup?action=edit` and add your Snipe-IT API key toset-up the app.
