Cado Response Integration

Cado Security provides a cloud-native digital forensics platform. By automating data capture and processing across cloud and container environments, Cado Response enables security teams to effectively investigate and respond to cyber incidents at cloud speed.

Getting Started

Please see the Integration documentation to get set up with Splunk SOAR and Cado Response.

Supported Actions

  • capture bucket - Capture contents of a specific S3 bucket processing and analysis on the Cado Response platform
  • capture instance - Capture a specific EC2 instance for processing and analysis on the Cado Response platform
  • loop pipeline - Get pipeline for specified pipeline ID on Cado Response platform until it is terminated
  • get pipeline - Get pipeline for specified pipeline ID on Cado Response platform
  • list pipelines - List pipelines for specified project on Cado Response platform
  • list buckets - List projects that are currently visible to the Cado Response platform
  • list instances - List EC2 instances that are currently visible to the Cado Response platform
  • create project - Create a project on Cado Response platform
  • list projects - List projects that are currently active on Cado Response platform
  • test connectivity - Validate the asset configuration for connectivity using supplied configuration
