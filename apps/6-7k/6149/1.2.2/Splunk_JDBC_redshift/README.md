## Splunk DBX Addon for RedShift

The Splunk [DBX](https://splunkbase.splunk.com/app/2686/) provides for the installation of JDBC Drivers using a Splunk Technology Add-on package (TA).
This TA package contains the JDBC Driver for [Redshift DB](https://aws.amazon.com/redshift/?whats-new-cards.sort-by=item.additionalFields.postDateTime&whats-new-cards.sort-order=desc)

# Binary File Declaration

redshift-jdbc42-2.1.0.19.jar is provided by a thirdparty source and releases can be found at [maven](https://mvnrepository.com/artifact/com.amazon.redshift/redshift-jdbc42)
