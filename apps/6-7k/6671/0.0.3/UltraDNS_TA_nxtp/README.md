## Event metadata

- Source: `ultradns_query://UltraDNS`
- Sourcetype: `ultradns`

## Configuration

On your Splunk instance navigate to `/app/UltraDNS_TA_nxtp` to perform the configuration.

**Configuration Settings**:

- **Username**: The Username which is used to access the API.
- **Password**: The Password which is used to access the API.
- **Lookback Days**: How many days to fetch data from the past.
