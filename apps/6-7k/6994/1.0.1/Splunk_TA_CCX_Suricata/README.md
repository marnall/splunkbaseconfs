**Description:**
The CCX Splunk Add-on for Suricata looks to provide a single field extraction bundle for Suricata EVE JSON logs ingetsed via Syslog.
This TA was built using a large dataset and endeavours to be the most CIM compliant comprehensive field extraction TA avaliable for Suricata EVE JSON logs ingetsed via Syslog.

Fully compatible with Splunk Enterprise and Splunk Cloud, built by an Ops team for Ops teams.

**Features:**
- This TA currently supports logtypes tagged under the following CIM datamodels: Alerts, Network Resolution, Network Session, Network Traffic, Web, Intrusion Detection (IDS), and Certificates (TLS).

   
**Compatibility:**

| --- | --- |

**Requirements:**
- This Add-on is intended to be installed on Splunk Search Heads and Forwarders.

**Installation:**
- Splunk Search Heads and Forwarders (Suricata EVE JSON logs via Syslog)

- Main sourcetype to be selected as part of monitoring configuration on Splunk Forwarder: "suricata"
