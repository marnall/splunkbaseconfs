The Conceal Splunk App records and reports all Conceal log and event data. The app is integrated with the following services:
 - ConcealBrowse
Conceal updates this app and enhances functionality as new field types and data feeds become available.

##### Searches and Alerts #####
This App used saves searches and come with preconfigured Alerts (disabled by default). The separate TA-conceal-add-on maps fields to the Splunk CIM data-model

###### Configuration ######
Install this App and Conceal Add-on in Standalone deployments
For Indexer cluster, install Add-on in Indexers
