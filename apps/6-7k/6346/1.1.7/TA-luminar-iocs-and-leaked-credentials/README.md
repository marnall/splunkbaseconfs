# Luminar IOCs & leaked credentials
Cognyte is a global leader in security analytics software that empowers governments and enterprises with Actionable Intelligence for a safer world. Our open software fuses, analyzes and visualizes disparate data sets at scale to help security organizations find the needles in the haystacks. Over 1,000 government and enterprise customers in more than 100 countries rely on Cognyte’s solutions to accelerate security investigations and connect the dots to successfully identify, neutralize, and prevent threats to national security, business continuity and cyber security.

Luminar is an asset-based cybersecurity intelligence platform that empowers enterprise organizations to build and maintain a proactive threat intelligence operation that enables to anticipate and mitigate cyber threats, reduce risk and enhance security resilience. Luminar enables security teams to define a customized, dynamic monitoring plan to uncover malicious activity in its earliest stages on all layers of the Web.

This connector allows integration of intelligence-based IOC data and customer-related leaked records identified by Luminar.

#Prerequisites

1. Luminar API Account ID
2. Luminar API Client ID
3. Luminar API Secret Key

## How to get Cognyte Luminar Credentials
- For registration and more details please contact luminar@cognyte.com

## Setup Process
1. Go to 'Apps' < 'Manage Apps'.

2. Find the 'Luminar IOCs & leaked credentials' app, and then click 'Launch 
   App'.

3. Once taken to the app, click on the 'Input' link in the navigation. Click 
   on "Create New Input"

4. Enter your credentials in popup window.

5. Following details user needs to provide:-

   - **Name:-** Enter the name of your Input.
   - **Interval:-** Interval value decide that at what interval of time your 
     app will run. Interval field take value in seconds.
   - **Index:-** By default index will be main. If user want to give there 
     own index they can change the Index value.
   - **Luminar API Account id:-** User needs to provide their Luminar API 
     Account ID.
   - **Luminar API Client ID:-** User needs to provide their Luminar API 
     Client ID.
   - **Luminar API Client Secret Key:-** User needs to provide their Luminar API 
     Client Secret Key.
   - **Luminar Initial Fetch Date:-** User needs to provide Initial Fetch date to pull the records.
    


# Use Case

Luminar IOCs and Leaked Credentials provide a default Dashboard. On 
Dashboard user can check following details:- 
1. Distribution of indicators over time.
2. Leaked Credentials count over time.
3. Leaked Credentials by Incident.

- **Note**: If the user provides a  new index for this integration during setup, they need to select the same
  index in the dropdown, which will be provided at the top of the dashboards.
