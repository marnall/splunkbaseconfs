## Splunk DBX Addon for Postgres

The Splunk [DBX](https://splunkbase.splunk.com/app/2686/) provides for the installation of JDBC Drivers using a Splunk Technology Add-on package (TA).
This TA package contains the JDBC Driver for [Postgres DB](https://jdbc.postgresql.org/)

# Binary File Declaration

postgresql-42.6.0.jar is provided by a thirdparty source and releases can be found at [maven](https://mvnrepository.com/artifact/org.postgresql/postgresql)
checker-qual-3.5.0.jar is provided by a thirdparty source and releases can be found at [maven](https://mvnrepository.com/artifact/org.postgresql/postgresql)
