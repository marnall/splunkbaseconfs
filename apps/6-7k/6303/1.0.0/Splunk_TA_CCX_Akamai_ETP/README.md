**Description:**

The CCX Add-on for Akamai ETP looks to provide the most CIM compliant comprehensive field extraction TA available for Akamai ETP ingested logs via syslog.

Akamai logtypes supported in this TA:  

- akamai:etp:aup
- akamai:etp:dns
- akamai:etp:threat


Fully compatible with Splunk Enterprise and Splunk Cloud, built by an Ops team for Ops teams.

**Features:**

- This TA currently supports logtypes tagged under the following CIM datamodels: Web, Intrusion Detection (IDS), Network Resolution (DNS), and Network Traffic.
