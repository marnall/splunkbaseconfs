# Picus Security App For Splunk

## Description

This app, developed by Picus, allows users to visualize data collected by Picus API within the Splunk platform. To use the app, users must first install the Picus Add-on for Splunk.

The app includes the following dashboards:
- Environment and Inventory
- Score

#### Configuration
To configure the app, users should update the picus_index macro located in Settings > Advanced Search > Search macros.

#### Environment and Inventory Dashboard
This dashboard includes the following panels:

- Simulation and Integration Agent Status
- Simulation and Integration Agent List
- Integration Health Status
- Integration List
- Simulation Count
- Simulation Status

#### Score Dashboard
This dashboard includes the following panels:

- Overall Prevention Results
- Overall Detection Results
- Mitigation Statistics by Vendor
