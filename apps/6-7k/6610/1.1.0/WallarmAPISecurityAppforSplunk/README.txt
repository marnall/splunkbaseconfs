Wallarm API Security App for Splunk Enterprise
===============================

Wallarm API Security App for Splunk is an application for Splunk Enterprise provided by Wallarm. You can use the application for the following:

- Get and analyze data on malicious requests that were made to your applications 
- Analyze vulnerabilities found in your applications
- Receive alerts about events generated by Wallarm API Security triggers
- Receive alerts about Wallarm service events, such as a new account added to your Wallarm personal account, changing integration settings with a third-party service, etc.


Software requirements
=====================

Wallarm API Security App for Splunk has the following software requirements:

- Splunk Enterprise 9.0 or later


Online documentation

For information about installing, configuring, running and troubleshooting Wallarm API Security App for Splunk, see online documentation located at https://docs.wallarm.com/user-guides/settings/integrations/splunk/
