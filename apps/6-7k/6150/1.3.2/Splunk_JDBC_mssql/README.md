## Splunk DBX Addon for Microsoft SQL Server

The Splunk [DBX](https://splunkbase.splunk.com/app/2686/) provides for the installation of JDBC Drivers using a Splunk Technology Add-on package (TA).
This TA package contains the JDBC Driver for [Microsoft SQL Server](https://docs.microsoft.com/en-us/sql/connect/jdbc/download-microsoft-jdbc-driver-for-sql-server?view=sql-server-ver15)

# Binary File Declaration

mssql-jdbc-12.4.1.jre8.jar is provided by a thirdparty source and releases can be found at [maven](https://mvnrepository.com/artifact/com.microsoft.sqlserver/mssql-jdbc)
