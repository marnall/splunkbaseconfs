Broadcom Compliance Event Manager for z/OS can send any of the monitored
events to Splunk for additional monitoring and/or alerting. Broadcom
Compliance Event Manager can be used to monitor the following list of
events (and more) for all three External Security Managers for z/OS,
Broadcom ACF2, Broadcom Top Secret, as well as IBM RACF, such as:

- User administration
- Policy administration
- Control option administration
- IPL member changes
- USS file changes
- TCPIP connections

The Broadcom Compliance Event Manager App for Splunk provides pre-configured
reports and dashboards to easily isolate and visualize Mainframe security
event data sent from Compliance Event Manager to Splunk.
