# Lumu add-on for Splunk
The Lumu Add-on for Splunk allows customers to poll and push adversary-related events to their Splunk deployments.

After installing and configuring the application, your Splunk deployment will be able to receive and parse adversarial activity detected by Lumu, allowing you to have visibility for your Security Operations team.

## Key components

### Lumu data model

This model was created using the data model feature present in the settings section of Splunk.

### Lumu dashboard

The Lumu dashboard was added using the **Dashboard** feature found in the **Search & Reporting** app.
