Dashboard
---------
This app was built with a PowerWalker VI 2200 LCD UPS as a data source and due to how Nut works and provides different values/fields for different UPS's the basic dashboard in this app may need some tinkering to reflect your setup.  i.e. Battery Voltage may differ and you will probably have to change the colour ranges on the filler gauge.

The dashboard makes us of a single base search with sub searches populating the panels.


WHERE TO INSTALL THIS TA
------------------------
This app only needs to be installed on Search Heads.
