# ABOUT THIS APP

- The Technology Add-on for Tenable is used to download web app scanning(was) vulnerability data from Tenable.io, do parsing and indexing on it.
- This is an add-on powered by the Splunk Add-on Builder.
- This Add-on uses Splunk KV store for checkpoint mechanism.

# REQUIREMENTS

- Splunk version and 9.0-9.3
- Appropriate API Key for collecting data from Tenable VM
- Python version: python3
