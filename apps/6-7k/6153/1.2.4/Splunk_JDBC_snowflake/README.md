## Splunk DBX Addon for Snowflake

The Splunk [DBX](https://splunkbase.splunk.com/app/2686/) provides for the installation of JDBC Drivers using a Splunk Technology Add-on package (TA).
This TA package contains the JDBC Driver for [Snowflake DB](https://docs.snowflake.com/en/user-guide/jdbc.html)

# Binary File Declaration

snowflake-jdbc-3.14.1.jar is provided by a thirdparty source and releases can be found at [maven](https://mvnrepository.com/artifact/net.snowflake/snowflake-jdbc)
