# ABOUT THIS APP

- The Technology Add-on for Tenable is used to download data from Tenable.asm, do parsing and indexing on it.
- This is an add-on powered by the Splunk Add-on Builder.

# REQUIREMENTS

- Splunk version and 9.0-9.3
- Appropriate API Key for collecting data from Tenable.asm
- Python version: python3
