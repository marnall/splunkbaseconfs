# Picus Security Add-on for Splunk

## Description

The Picus Security Add-on for Splunk allows users to collect data from the Picus Platform via Picus API. Users can collect simulations, agents and integrations as well as their details. After collecting, users can leverage this data by running custom searches or visualising it using the Picus App for Splunk.

#### Details

This add-on leverages the following Picus API Endpoints:
- Agent List, Agent Detail
- Integrations, Integration Agents
- Mitigation Device List
- Simulation List, Simulation Detail
- Latest Simulation Run Result
