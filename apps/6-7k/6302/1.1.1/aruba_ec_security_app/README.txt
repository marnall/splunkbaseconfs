
HPE Aruba Networking EdgeConnect SD-WAN Security App for Splunk
=================================

* **App Homepage:** https://splunkbase.splunk.com/app/6302
* **Authors:** Jaffer Sadhiq - HPE Aruba Networking
* **App Version:** 1.1.1

### Description ###

The HPE Aruba Networking EdgeConnect SD-WAN Security App for Splunk provides a dashboard view of all security event 
notifications exported from EdgeConnect devices within an enterprise’s SD-WAN. 
Customers can easily configure any EdgeConnect SD-WAN device (including virtualized and cloud-hosted) 
to forward security event notifications to Splunk, 
providing a convenient way to centralize logging, visualization, 
and analysis of security events alongside other telemetry or network events. 

From Splunk, users can filter, sort, navigate, and view the collective security event notifications generated across the SD-WAN, 
overall trends, and top talkers, to help customers pinpoint network events that warrant further investigation.  
The HPE Aruba Networking EdgeConnect SD-WAN Security App for Splunk requires ECOS release 9.1 or later.
