**Description:**
The CCX Extensions Add-on for ExtraHop looks to provide a single field extraction bundle for ExtraHop logs.
This TA was built using a large dataset and endeavours to be the most CIM compliant comprehensive field extraction TA available for ExtraHop.

Fully compatible with Splunk Enterprise and Splunk Cloud, built by an Ops team for Ops teams.

**Features:**
- This TA currently supports logtypes tagged under the following CIM datamodels: Alert, IDS, Malware, and Network Traffic.

| --- | --- |
