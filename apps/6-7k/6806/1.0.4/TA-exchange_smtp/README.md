### About

This technology add-on (TA) is for collecting and parsing Microsoft Exchange SMTP logs.

### Installation

This TA is meant to be installed on both the Microsoft Exchange Server where the logs are to be read from, and on the Splunk search heads for parsing. The input stanza can be disabled on the copy of the TA installed on the search heads.

### Configuration

Make sure that the input path for the SMTP logs is correct and set a proper index before enabling the input.
