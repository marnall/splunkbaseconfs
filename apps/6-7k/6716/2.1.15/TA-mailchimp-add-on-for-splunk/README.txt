Overview
This Add-on provides valuable insights on informations added in your mailchimp account.

Mailchimp Add-on leverages data from Mailchimp to Splunk via API Call. It uses multiple API to fetch data from Mailchimp such as information about account , audience, campaigns and reports.
Get the Activation key by visiting https://avotrix.com/splunkapp.php
