
# Splunk DBX Addon for Oracle

The Splunk [DBX](https://splunkbase.splunk.com/app/2686/) provides for the installation of JDBC Drivers using a Splunk Technology Add-on package (TA).
This TA package contains the JDBC Driver for [Oracle DB](https://www.oracle.com/database/technologies/appdev/jdbc.html)

# Binary File Declaration

lib/dbxdrivers/ojdbc8-23.2.0.0.jar is provided by Oracle source and releases can be found at [maven](https://mvnrepository.com/artifact/com.oracle.database.jdbc/ojdbc8)
