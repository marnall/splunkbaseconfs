PingFederate App for Splunk 2.1.2
---------------------------------

Introduction
------------
The PingFederate App for Splunk gathers and presents transaction metrics from PingFederate via a series of custom
reports and graphical illustrations.

The application enables identity and access management (IAM) administrators,
architects, and security managers to easily obtain custom reports for all PingFederate log data. Administrators, can view each 
authentication event per app and authentication source, and analyze these event data over time. 

The custom reports display key events across account management, Identity Provider, Service Provider and 
OAuth Authorization Server transactions.
