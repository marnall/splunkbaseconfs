Welcome to Splunk for Cisco Security Application
The Splunk for Cisco Security Application gives you a single interface to manage all of your Cisco Security Add-ons .
The app requires you have the one or more of the supported Add ons installed.

  • Splunk for Cisco Firewalls (ASA or PIX)
    http://www.splunkbase.com/apps/All/4.x/Add-On/app:Cisco+Firewalls+Add-On

  • Splunk for Cisco IPS
    http://www.splunkbase.com/apps/All/4.x/Add-On/app:Cisco+IPS+SDEE+Data+Collector

  • Splunk for Cisco IronPort Web Security
    http://www.splunkbase.com/apps/All/4.x/App/app:Cisco+IronPort+Web+Security+Application


  • Splunk for Cisco IronPort E-Mail Security
    http://www.splunkbase.com/apps/All/4.x/Add-On/app:Cisco+IronPort+E-mail+Security+Add+On

  • Splunk for Cisco Client Security Agent
    http://www.splunkbase.com/apps/All/4.x/Add-On/app:Cisco+Client+Security+Agent+Add+On
