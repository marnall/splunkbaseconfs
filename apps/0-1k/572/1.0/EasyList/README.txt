This app provides one command, easylistfilter, which will remove entries from the results that contain URLs to advertisements. 

It uses the publicly available EasyList ad server list, which is used by AdBlock. It is not a complete reimplemntation of AdBlock, so doesn't take advantage of all of the rules in the list. It is also not terribly fast, but is functional.

Requirements:
At this point, the download script is a shell script and requires curl, so either UNIX or cygwin is required.

Let me know if you find it useful or need some help.
