Description
This Splunk for Weather Incident Tracking App is an example of how you can use Splunk analyize the strom such as tornado, hail and damaging wind. This app is the first edition to analyze weather incident, and the prediction function will be shown in the next edition. Here you'll find dashboards and reports for the tornado, hail and damaging wind of your interest.

General Features
  • Weather Incidenthistory
There is some analysis for tornado, hail and damaging wind each year, it can help us to find the relationship between those weather phenomena.

torn analysis
This dashboard shows the analysis between tornado and year, month, time and state, etc. It can help us to analyze the possible occurence of tornado phenomena.
hail analysis
This dashboard shows the analysis between hail and year, month, time and state, etc. It can help us to analyze the possible occurence of hail phenomena.
wind analysis
This dashboard shows the analysis between damaging wind and year, month, time and state, etc. It can help us to analyze the possible occurence of damaging wind phenomena.
