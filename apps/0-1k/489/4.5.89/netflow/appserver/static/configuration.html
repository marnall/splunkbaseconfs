NFO Logic Modules

Get more value out of NetFlow by enabling additional NetFlow Optimizer (NFO) inline analytics (Logic Modules). Once enabled and configured, the corresponding Splunk App dashboards will show NetFlow Analytics data for your desired use cases.

By default NetFlow Optimizer is preconfigured with one Logic Module enabled – “10067: Top Traffic Monitor” packaged in NFO “Network Traffic and Devices Monitor” Module Set. You may enable / disable the entire Module Set or each Module by clicking on / .

To learn more about NetFlow Optimizer Modules please visit https://docs.netflowlogic.com/netflow_optimizer/user-guide/solutions-at-a-glance/.
