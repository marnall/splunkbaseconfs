Overview

Splunk Particle is an experiment in visualizing Splunk real-time search results.

Read more about this on the main project page or create your own by starting on the base template.

Included examples

Twitter client activity - see the distribution of clients connecting to Twitter as seen from the sample stream. The most common clients have icons associated with them.

Tagged tweets - see the tweets that have hashtags, their client type, and the raw tweet.

Splunk indexing lag - see how long it takes for data to show up in Splunk, the host it came from, and the breakdown between data that is and is not eventtyped.

Tcpdump traffic - see real-time network traffic from the packet level, split by network layer, protocol, and hardware vendor.

Splunk source pianoroll - see how Splunk receives data in both streamed and buffered-write mode.
