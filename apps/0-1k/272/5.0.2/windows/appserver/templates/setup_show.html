Overview

The Splunk App for Windows provides pre-built data inputs to facilitate monitoring Windows infrastructure using Splunk. Check out the Splunk for Windows page on Splunkbase for support information, the latest updates, and more.

Configuration of inputs through this application will make these inputs visible to other Splunk applications on this system. After configuration, confirm that the changes you make in this application do not negatively alter the other applications.

Note that configuration of inputs through this application will only affect data collection on this machine. To collect data from other machines in your environment and report on them through this application, please refer to this documentation - http://docs.splunk.com/Documentation/WindowsApp/5.0/User/WhataSplunkAppforWindowsdeploymentlookslike.
